# WebRTC-Android-Learn

Read The Fucking WebRTC Code. 针对Android方向，阅读源代码，添加注释，并适当改写。master分支稳定，develop分支可能出错但包含更多新特性。

改写部分包括：

    支持Android端屏幕录像。
