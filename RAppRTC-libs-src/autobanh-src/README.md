### 说明

AutobahnAndroid 是 Android 平台上的 WebSocket & WAMP (WebSocket Application Messaging Protocol) 实现。

### 特点：

- WebSocket 和 WAMP 客户端
- 实现了 WebSocket RFC6455, Draft Hybi-10+ and WAMP v1
- 支持 Android 2.2+
- 良好遵循标准
- 高性能异步设计
- 易用的 API
- 无缝集成 Android UI 应用
- UI 线程上无网络 Activity

### 使用

在WebSocketChannelClient中使用。
